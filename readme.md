# CPU Scheduler project

>CPU 스케줄러 프로젝트입니다  
>구현한 알고리즘은 다음과 같습니다  

```
1.SJF Scheduling  
2.Priority Scheduling  
3.Round Robin Scheduling  
4.Own Scheduling(Round Robin Scheduling 기반 TimeQuantum 설정 방식)  
```


branch로는 testFile이 있습니다. testFile의 txt는 다음과 같이 생겼습니다.

```
test1.txt
Process 1 0 7
Process 2 5 4 
Process 3 6 1 
Process 4 8 4
```
